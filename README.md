# Student Grade Calculator 🎓

## Project Description
This is a comprehensive Python application built during **Week 2** of my internship at **The Developers Arena**. The program allows educators to input marks for multiple students across various subjects, automatically calculates their averages and grades, and provides personalized feedback. It also generates a summary report including the class average and identifies the top and bottom performers.

## Key Features
- **Multi-Student Processing:** Handles data for any number of students using loops.
- **Automated Grading:** Uses a robust `if/elif/else` system to assign grades (A-F).
- **Class Statistics:** Automatically calculates the highest, lowest, and average scores for the entire group.
- **Input Validation:** Implements `try-except` blocks and `while` loops to prevent errors from invalid marks or non-numeric inputs.
- **Formatted Reporting:** Displays results in a clean, tabular format for easy reading.

## Technical Skills Used
* **Control Flow:** Conditional logic for grading and loops for data iteration.
* **Data Structures:** Lists and Dictionaries to store and organize student records.
* **Error Handling:** Robust validation to ensure data integrity.
* **Functions:** Modular code structure for reusability.
